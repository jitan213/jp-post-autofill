# 国際郵便マイページ 住所オートフィル

eBayの注文画面からコピーした住所ブロック（名前〜電話番号）を貼り付けるだけで、
国際郵便マイページ（スマホ版含む）のお届け先入力フォームに自動入力するChrome拡張です。

## インストール（初回のみ）

1. このフォルダ（jp-post-autofill）をzipから解凍して、消さない場所に置く（例: 書類フォルダ）
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」→ 解凍したフォルダを選択
5. ツールバーのパズルアイコン → ピン留めしておくと便利

## 使い方

1. 国際郵便マイページの「お届け先入力」画面（Step2）を開く
2. ツールバーの拡張アイコンをクリック
3. eBayの「お届け先」をコピーして、一番上の欄に貼り付け
   ```
   Peter Cooper
   8 Lafachie
   16150 Exideuil-sur-Vienne
   France
   +33 6 81 23 86 59
   ```
4. 名前・住所・州名・郵便番号・国名・電話番号に自動で振り分けられるので、
   間違いがあればここで修正
5. 「フォームに入力」を押す → ページの各欄に入る
6. 内容を目視確認してから「次へ」を押す（送信は自動では行いません）

## 対応している解析

- 電話番号: どの行にあっても自動検出（+33… などの国際形式）
- 国名: 主要国を英語表記で判定し、プルダウンから自動選択
  （USA → UNITED STATES OF AMERICA などの表記ゆれも吸収）
- 郵便番号: 国ごとの形式に対応（英国 NW1 6XE、カナダ K1A 0B1、
  蘭 1234 AB、ポーランド 00-001、ブラジル 01310-100 など）
- 州名: 米国・カナダ・豪州の略称（NY / ON / NSW など）を自動抽出
- 30文字/行の制限に合わせて自動改行（名前3行・住所6行まで）

## うまく入らないとき

- 「ページと通信できません」→ マイページのタブを一度リロードしてから再実行
- 国名プルダウンが選択されない → 手動で選択（ステータスに表示されます）
- 郵便局側の画面構成が変わった場合はラベル名の変更が原因の可能性があります
